# HappyClient

HappyClient é um sistema web desenvolvido como MVP acadêmico para gerenciamento de clientes e controle de aniversários. O projeto foi construído utilizando Flask, SQLite e SQLAlchemy, permitindo o cadastro, consulta, edição e exclusão de clientes, além da visualização dos aniversariantes do dia.

## Funcionalidades

- Login de usuários
- Cadastro de clientes
- Listagem de clientes
- Pesquisa de clientes
- Edição de informações
- Exclusão de registros
- Cadastro de data de nascimento
- Tela de aniversariantes do dia

## Tecnologias Utilizadas

- Python
- Flask
- SQLAlchemy
- SQLite
- HTML
- Bootstrap

## Objetivo

O projeto foi desenvolvido com fins acadêmicos para aplicar conceitos de desenvolvimento web, persistência de dados, arquitetura CRUD e integração entre front-end e back-end utilizando Python.
